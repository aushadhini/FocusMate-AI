🚀 FocusMate AI <br>
A modern productivity web application designed to help users stay focused, manage tasks efficiently, and build consistent work habits using the Pomodoro technique.

🔗 Overview <br>
FocusMate AI combines task management, time tracking, and productivity analytics into a single intuitive platform. It enables users to transform small, focused sessions into long-term consistency and measurable progress.

✨ Key Features <br>

🔐 Authentication<br>
Secure user authentication powered by Supabase<br>
Persistent login sessions<br>

📝 Task Management<br>
Create, edit, and delete tasks<br>
Mark tasks as completed<br>
Smart task suggestions for productivity<br>

⏱ Focus Timer (Pomodoro)<br>
25-minute focus sessions<br>
Start, pause, and reset controls<br>
Automatic session tracking<br>

📊 Analytics & Insights<br>
Total focus sessions and minutes<br>
Daily activity tracking<br>
Current and best streak calculation<br>
Weekly performance visualization<br>
Activity heatmap<br>

🎯 Dashboard<br>
Real-time productivity overview<br>
Recent tasks display<br>
Motivation insights based on user progress<br>

🛠 Technology Stack<br>
Frontend<br>
React (Vite)<br>

Custom CSS (Modern UI Design)<br>
Backend & Database<br>
Supabase (Authentication + PostgreSQL Database)<br>
