# FocusMate AI

A full-stack productivity app built with React and Node.js.

## ✨ Features

- ➕ Add tasks
- ❌ Delete tasks
- ✏️ Edit tasks
- ☑️ Mark tasks as completed
- 🤖 AI-based task suggestions

## 🛠 Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Styling: CSS (inline)

## ⚙️ How to Run

### Backend
```bash
node server.js
